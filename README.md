## 安装项目

建议运行 npm，因为 cnpm 会出现一些未知问题

如果遇到 node-sass 报错：
解决方法：https://blog.csdn.net/weixin_40013817/article/details/105218844

### 启动项目
npm start

原生html项目在 src/html 目录下面，此目录下面文件是独立的。
